Предназначен для отображения родительских и подчиненных объектов.

Формируется командой Структура подчиненности на панели навигации документа.

Предоставляет оперативный и наглядный доступ ко всей цепочке взаимосвязанных с точки зрения бизнес-логики объектов. Объекты считаются взаимосвязанными, если один ссылается на другой. В программе взаимосвязь объектов выглядит так: в одном объекте (назовем его дочерним) в качестве какого-либо реквизита указан другой объект (назовем его родительским). Структура подчиненности позволяет из какого-либо одного объекта увидеть всю цепочку взаимосвязанных с ним объектов.

В форму попадают все дочерние и все родительские объекты текущего объекта. В наглядной форме представлена взаимосвязь этих объектов.

Объект, для которого сформирована структура подчинености, отображается жирным шрифтом.

Доступные действия

  • С помощью двойного щелчка мышью на любом выведенном объекте можно открыть данный объект для просмотра.
  • С помощью кнопки Обновить можно переформировать структуру подчиненности.
  • С помощью кнопки Вывести для текущего можно вывести Структуру подчиненности для выделенного объекта.
