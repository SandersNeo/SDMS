1. Просмотренные заявки. Содержит список заявок, которые открывались сегодня, либо открывались задачи, в которых основанием являются заявки. Кроме этого в список попадут заявки, созданные инструментом "Групповое создание заявок".

При открытии данного инструмента происходит автоматический выбор заявок для распределения трудозатрат (выбираются заявки, по которым не занесены трудозатраты за текущий день). В левой колонке есть возможность выбрать заявки, которые будут участвовать в распределении трудозатрат из блока "Распределяемая деятельность". После распределения трудозатрат в колонке "Распределено" будут отображаться трудозатраты, которые автоматически распределены по выбранным заявкам.

Заявки, в которые были занесены трудозатраты за текущий день выделены серым шрифтом. Занесенные трудозатраты отображаются в колонке Внесено. По нажатию кнопки добавления трудозатрат  в строке заявки можно добавить трудозатраты напрямую в заявку.


2. Создание заявки - позволяет быстро создать заявку с возможностью внесения трудозатрат. Для создания заявки требуется заполнить все поля. После заполнения кнопка "Создать" становится доступной. При нажатии на неё создается "Заявка на разработку" и связанная с ней "Задача". Задача будет последовательно переведена между статусами, от "Новая" до статуса, указанного в поле "Статус". В зависимости от выбранного статуса затраты времени из поля "Трудозатраты" будут добавлены в "Заявку" или "Задачу":
 - для статусов "Новый" или "В реализацию", трудозатраты будут добавлены в заявку;
 - для всех остальных статусов трудозатраты будут добавлены в задачу.

Можно отказаться от создания "Заявки", нажав кнопку Отменить.

3. Распределяемая деятельность. В этом блоке происходит распределение времени по видам деятельности. Состав списка видов деятельности настраивается по кнопке "Настройка". Для каждой строки можно указать затраты времени и нажать кнопку "Распределить". Указанные затраты времени будут равномерно распределены по отмченным заявкам.

При распределении трудозатрат действует ограничение на итоговое количество трудозатрат. Ограничение составляет 24 часа. При попытке распределить больше времени будет выдано предупреждение.
