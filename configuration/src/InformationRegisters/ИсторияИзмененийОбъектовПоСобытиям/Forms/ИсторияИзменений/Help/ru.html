Форма истории изменений объекта. Вы можете просмотреть историю изменений объекта. 


Кнопка "Настройки" предназначена для индивидуальной настройки истории.

На форме настроек доступен отбор по объектам и настройки упорядочивания.

Отбор по объектам позволяет установить по каким объектам будет отображаться история (для заявки на разработку попадут все подчиненные ей задачи)

Настройки упорядочивания предназначены для сортировки и выбора отображаемых событий. В категорию главное попадают следующие события:

  1. Создание объектов.
  2. Изменение статуса объекта.
  3. Изменение поле назначена (для тех объектов, у которых он существует).
  4. Изменение описания.

Все остальные события попадают в категорию прочие. Для применения настроек необходимо нажать кнопку "Сохранить".

Кнопка "Показать все изменения" на форме истории изменений показывает историю по всем объектам и всем категориям событий.
